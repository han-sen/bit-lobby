# Bit Lobby

Chat rooms for code talk.

Live demo: https://bitlobby.herokuapp.com/

## Tech Stack

-   node.js
-   mongodb
-   express.js
-   react view templates
-   socket.io

## Install

Clone into `bit-lobby`, run `npm i` to install dependencies, and `npm run dev` to start the server.
